Computes the world transformation matrix for the scene object.

Calculates the object's world transformation by combining its local transformation with its parent's world transformation. If the object has no parent, returns its local transformation matrix.

Returns

Note

void Engine::Scene::OnRender ( Renderer &  renderer)
virtual

Called every frame to render scene.

Renders the scene by rendering terrain and scene objects.

This method manages the rendering process for the scene. It first ensures the renderer is initialized by storing its reference and calling OnCreate if not already done. Then it renders the terrain system (if present) followed by rendering the root scene object and its entire hierarchy.

Parameters
rendererReference to the rendering system used to draw scene elements.
Note
If no renderer was previously set, this method will trigger scene initialization.
Terrain is rendered before scene objects to establish background rendering.

Renders an object and its children.

Recursively renders a scene object and its children.

Parameters
objectObject to render
rendererRenderer to use

This method traverses the scene graph, rendering each object with a valid vertex array and material. It prepares a render command with the object's world transformation matrix and submits it to the renderer. The method then recursively renders all child objects of the current object.

Parameters
objectA shared pointer to the SceneObject to be rendered.
rendererThe Renderer instance used to submit rendering commands.
Note
If the object is null or lacks a vertex array or material, it will be skipped.
The rendering is performed using the object's world transformation matrix.
