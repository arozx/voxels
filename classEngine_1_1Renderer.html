◆ Render()

void Engine::Renderer::Render ( )

Triggers the rendering process by flushing all queued render commands.

This method calls the Flush() method to process and execute all pending render commands in the render queue. It serves as a simple wrapper method to initiate the rendering pipeline.

Note
This method ensures that all queued rendering commands are processed and drawn to the screen.
