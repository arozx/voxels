void Engine::OpenGLWindow::Init ( const WindowProps &  props)
privatevirtual

Initializes GLFW window and OpenGL context.

Initialize the OpenGL window with specified properties.

Parameters
propsWindow properties
propsWindow configuration parameters including title, width, and height

Performs comprehensive window initialization:

  • Initializes GLFW library
  • Configures OpenGL context version and profile
  • Creates GLFW window
  • Sets up window context
  • Registers multiple input callbacks for window events, keyboard, mouse, and scroll interactions
Note
Terminates GLFW and logs an error if window creation fails
Enables vertical synchronization by default after initialization
Exceptions
NoneHandles initialization errors internally
